安裝與快速開始

從零開始，五分鐘內讓 GCD 跑起來

系統需求

  • Rust 工具鏈：1.75+ (stable channel)
  • 作業系統：macOS / Linux / WSL2
  • API 金鑰（至少一個）：GEMINI_API_KEY、OPENAI_API_KEY 或 ANTHROPIC_API_KEY

編譯安裝

terminal 
# 克隆專案
git clone https://github.com/voidful/GemiClawDex.git
cd GemiClawDex

# 編譯 release 版本（約 30-60 秒）
cargo build --release

# 產出的二進位檔
ls -la target/release/gcd

# 加入 PATH（可選）
export PATH="$PWD/target/release:$PATH"

設定 API 金鑰

~/.bashrc 或 ~/.zshrc 
# Gemini（免費額度最高）
export GEMINI_API_KEY="AIza..."

# OpenAI
export OPENAI_API_KEY="sk-..."

# Anthropic
export ANTHROPIC_API_KEY="sk-ant-..."

驗證安裝

terminal 
# 檢查版本與工作區資訊
gcd overview

# 檢查供應商狀態
gcd providers doctor

# 列出所有可用供應商
gcd providers list

第一次執行

terminal 
# 進入任何專案目錄
cd ~/my-project

# 啟動互動式 REPL
gcd

# 或執行單次任務
gcd exec "解釋這個專案的架構"
小提示

GCD 會自動偵測專案根目錄（透過 .git/、Cargo.toml、package.json 等標記檔案），無需手動指定工作區。
