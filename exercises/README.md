# GCD 動手練習

這些練習配合教學網站的章節設計。每個練習都有明確的目標、步驟和預期結果。

## 前提條件

- 已完成 GCD 的編譯安裝（參考教學網站 U1）
- 至少設定一個 API 金鑰（建議用 Gemini 免費額度）

## 練習列表

| 練習 | 對應章節 | 難度 | 預估時間 |
|------|---------|------|---------|
| 01 — 第一次使用 Agent | 入門範例 | 🟢 | 10 分鐘 |
| 02 — 撰寫 AGENTS.md | AGENTS.md 工作坊 | 🟢 | 20 分鐘 |
| 03 — Context 注入對比 | Context Engineering | 🟡 | 15 分鐘 |
| 04 — 建立自訂命令 | 專案規範、命令和技能包 | 🟡 | 15 分鐘 |
| 05 — 建立可重用 Skill | 專案規範、命令和技能包 | 🔴 | 30 分鐘 |

## 使用方式

每個練習目錄包含：
- `README.md` — 題目說明和步驟
- `solution.md` — 參考答案（建議先自己做完再看）
- 其他輔助檔案（範例專案、模板等）
